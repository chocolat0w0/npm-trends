# npm-trends

npm-trends は、複数パッケージの直近 1 年分ダウンロード数を同じ折れ線グラフで比較できるツールです。

## できること

- パッケージ名を追加すると、365 日分のダウンロード推移が 1 週間ごとの数値で描画されます。
- URL クエリに比較中のパッケージが含まれるので、リンク共有だけで同じ条件を再現できます。
- ラベルをクリックすると不要なパッケージをすぐに削除できます。

## 開発メモ

- 動作環境: Node.js 24
- セットアップ: `npm install`
- 開発サーバー: `npm run dev` → ブラウザで `http://localhost:5173`
- 本番ビルド: `npm run build` / プレビュー: `npm run preview`
